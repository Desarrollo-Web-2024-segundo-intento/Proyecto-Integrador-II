Estudios médicos de baja, mediana y alta complejidad.

Baja complejidad:

* Análisis de sangre para medir los niveles de glucosa en pacientes con diabetes.

* Radiografía de tórax para detectar neumonía o fracturas de costillas.

* Ecografía abdominal para evaluar el hígado, el riñón y la vesícula biliar.

* Electrocardiograma (ECG) para evaluar la actividad eléctrica del corazón.

* Prueba de esfuerzo en cinta rodante para evaluar la función cardíaca durante el ejercicio.

Mediana complejidad:

* Tomografía computarizada (TC) de abdomen para diagnosticar apendicitis aguda.

* Colonoscopia para detectar pólipos o cáncer colorrectal.

* Resonancia magnética (RM) de la columna vertebral para evaluar hernias discales.

* Endoscopia digestiva alta para diagnosticar úlceras estomacales o esofagitis.

* Cateterismo cardíaco para evaluar el flujo sanguíneo en las arterias coronarias.

Alta complejidad:

* Trasplante de médula ósea para tratar la leucemia.

* Cirugía de bypass coronario para restaurar el flujo sanguíneo al corazón.

* Radioterapia externa para tratar el cáncer de próstata.

* Quimioterapia intensiva para el tratamiento del linfoma no Hodgkin.

* Implante coclear para restaurar la audición en pacientes con sordera profunda.
